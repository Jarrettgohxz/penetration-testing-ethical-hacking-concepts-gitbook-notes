# 4. System enumeration (via shell)

### 4.1 General enumeration

...

```shellscript
# architecture
$ uname -u
$ cat /etc/version
$ cat /etc/banner
$ cat /etc/issue

$ nvram get firmware_version
$ nvram show | grep -i version
 

```

### 4.2 `httpd` binary (CVE)

...

```shellscript
# "httpd" binary
$ which httpd
$ find / -type f -name '*httpd*' 2>/dev/null

$ cat /etc/inittab
$ grep -R "httpd" /etc/init.d 2>/dev/null
```
