# Bloodhound

A GUI tool that works together with the enumeration tool _Sharphound_. Generally, _Sharphound_ will be executed to enumerate through the AD environment to collect information, before being passed to _Bloodhound_ to be displayed visually. Essentially, _Sharphound_ can be viewed as the data collector for _Bloodhound_.

> Note: the Bloodhound and Sharphound versions should match for the best results.
